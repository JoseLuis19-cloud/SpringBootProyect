
	Pasos para instalar la aplicación en la máquina Server
	------------------------------------------------------
	
	1. Instalar Tomcat 9 y configurarlo como 
	   El EAR WAR generado copiarlo en  ....
	
	2. Instalar My SQL 8 y configurar base de datos y usuario con password.

	3. Ejecutar Script SQL de creación de tablas de la aplicacion.
	
	4. Ejecutar Script SQL de carga inicial de datos y usuario Administrador
	   con las opciones básicas para dar de alta menus y menus de usuario.
	
	5. Crear directorios facturas e informes en el directorio Windows o Unix,
	   donde se generarán las facturas y PDFS. En MAC será  /Users/Administrador/Documents/ 
 
       /Users/UsuarioJoseLuis/Documents/Informes/
	   C:/Users/Public/Informes/

	   /Users/UsuarioJoseLuis/Documents/Facturas/
	   C:/Users/Public/Facturas/
	  
	 6. Crear directorios copiaseguridad en el directorio Windows o Unix,
	    donde se generarán las copias de seguridad En MAC será /Users/Administrador/Documents/CopiaSeguridad 
 
       /Users/UsuarioJoseLuis/Documents/CopiaSeguridad/
	   C:/Users/Public/CopiaSeguridad/

	 7. Crear directorios log4j2 en el directorio Windows o Unix,
	    donde se generarán los fichero logs de la aplicacion
 
       /Users/UsuarioJoseLuis/Documents/logs/
	   C:/Users/Public/logs/
